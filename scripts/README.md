# Скрипты проекта

## download-sexolog-gifs.js (Основной скрипт)

Парсер для скачивания GIF файлов с сайта sexolog.jofo.me (страница с камасутрой).

### Использование

```bash
npm run download-gifs
```

### Что делает скрипт:

1. Загружает HTML страницу `https://sexolog.jofo.me/1100175-kamasutra-animashki.html`
2. Парсит HTML и находит все ссылки на GIF файлы в `<img src>` тегах внутри `div.post_wrapper_single`
3. Создает папку `public/gifs-sexolog/` (если её нет)
4. Скачивает все найденные GIF файлы
5. Сохраняет их локально в `public/gifs-sexolog/`
6. Пропускает уже существующие файлы (можно запускать повторно)
7. Показывает прогресс и статистику скачивания

### Преимущества:

- **Автоматический парсинг** - находит все GIF на странице автоматически
- **Большая коллекция** - скачивает множество различных поз камасутры
- **Независимость от внешнего сервера** - GIF работают даже без интернета
- **Быстрая загрузка** - файлы отдаются с локального сервера
- **Повторное использование** - можно запускать скрипт несколько раз, он пропустит уже скачанные файлы

### Примечания:

- Папка `public/gifs-sexolog/` добавлена в `.gitignore`, чтобы не коммитить большие файлы в репозиторий
- Сервер использует GIF из папки `public/gifs-sexolog/` как основную (доступны через `/gifs/`)
- Скрипт делает небольшую задержку (200ms) между запросами, чтобы не перегружать внешний сервер
- Если структура страницы изменится, может потребоваться обновление парсера

## download-gifs.js (Старый скрипт)

Скрипт для скачивания GIF файлов с внешнего сервера fanty-online.com (устаревший).

### Использование

```bash
npm run download-gifs-old
```

### Что делает скрипт:

1. Создает папку `public/gifs/` (если её нет)
2. Скачивает все GIF файлы от `poza-1.gif` до `poza-65.gif` с `https://fanty-online.com/data/uploads/`
3. Сохраняет их локально в `public/gifs/`
4. Пропускает уже существующие файлы (можно запускать повторно)
5. Показывает прогресс и статистику скачивания

### Примечания:

- Этот скрипт устарел, рекомендуется использовать `npm run download-gifs` (новый парсер)
- Старые GIF доступны через `/gifs-old/` для совместимости
- Скрипт делает небольшую задержку (100ms) между запросами
