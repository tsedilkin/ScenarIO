# Скрипты проекта

## download-gifs.js

Скрипт для скачивания всех GIF файлов с внешнего сервера и сохранения их локально в проекте.

### Использование

```bash
npm run download-gifs
```

### Что делает скрипт:

1. Создает папку `public/gifs/` (если её нет)
2. Скачивает все GIF файлы от `poza-1.gif` до `poza-65.gif` с `https://fanty-online.com/data/uploads/`
3. Сохраняет их локально в `public/gifs/`
4. Пропускает уже существующие файлы (можно запускать повторно)
5. Показывает прогресс и статистику скачивания

### Преимущества:

- **Независимость от внешнего сервера** - GIF работают даже без интернета
- **Быстрая загрузка** - файлы отдаются с локального сервера
- **Надежность** - не зависит от доступности внешнего сайта
- **Повторное использование** - можно запускать скрипт несколько раз, он пропустит уже скачанные файлы

### Примечания:

- Папка `public/gifs/` добавлена в `.gitignore`, чтобы не коммитить большие файлы в репозиторий
- Если локальный файл не найден, сервер автоматически использует внешний URL как fallback
- Скрипт делает небольшую задержку (100ms) между запросами, чтобы не перегружать внешний сервер
